\# Fraud Detection Dashboard



Advanced multi-page Streamlit dashboard for fraud transaction analysis.



\## Features



\- Fraud Detection Overview

\- Transaction Explorer

\- SHAP Fraud Explainer

\- Interactive Plotly Charts

\- Sidebar Filters

\- Live Risk Score



\## Tech Stack



\- Python

\- Streamlit

\- Plotly

\- SHAP

\- Pandas

\- NumPy

\- Scikit-learn



\## Live Dashboard

The original dataset files (train_transaction.csv and train_identity.csv) were too large for GitHub upload.

This project used the IEEE-CIS Fraud Detection dataset from Kaggle.
